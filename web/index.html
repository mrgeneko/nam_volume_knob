Analytics Disclosure

This app uses Google Analytics 4 to track usage and errors. The following data is collected:

  • Processing Events: File names processed, model architecture (A1/A2), NAM version, gain levels applied
  • Error Events: Error type, error message, file name that caused the error
  • Batch Summaries: Number of files processed, number of gains applied, A1 vs A2 model distribution
  • Standard Analytics: Page views, browser type, device type, geographic location (from IP)

What is NOT collected: Audio data, model weights, file contents, personal information

If you deploy this tool to your own URL, update the tracking ID in index.html (line 12) to send analytics to your own Google Analytics property instead of the original author's.
